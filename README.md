## Setup

```sh
git clone https://github.com/GPT-Engineer-App/nextjs-payload3-blog-types.git
cd nextjs-payload3-blog-types
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
